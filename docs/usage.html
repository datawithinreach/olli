HTML <script> Tags

The most basic way to use Olli is to add it to any basic HTML page.

1. Add the script tag inside the document <head>:

<html> .... <head> .... <script src="https://.../olli.js" /> <script src="https://.../olli/VegaLiteAdapter.js" /> </head> </html> 2. Call Olli from a <script> tag: .... <script> olli({ visualization: vegaLiteAdapter(visSpec, additionalInfo), renderType: 'tree' domId: 'Accessible-Vis' }) </script>

JavaScript Applications

Olli can also be easily added to normal JavaScript applications.
1. Install the preset: npm install olli 2. Import `olli`, and the adapter you want to use, into the file you want to use it in: import { olli } from 'olli'; import { VegaLiteAdater } from 'olli/VegaLiteAdater' 3. Call `olli` and set-up your configuration object: ... olli({ visualization: VegaLiteAdapter(visSpec, additionalInfo), renderType: 'tree' domId: 'Accessible-Vis' }) ...

React

Olli can even be used with React, but due to how React handles the state of the DOM, `olli` has to be called inside the `useEffect(() => {}) hook. Install preset: npm install olli 2. Import `olli`, and the adapter you want to use, into the component you want to use it in import { olli } from 'olli'; import { VegaLiteAdater } from 'olli/VegaLiteAdater' 3. Call `olli` and set-up your configuration object inside the `useEffect(() => {})` hook: ... useEffect(() => { olli({ visualization: vegaLiteAdapter(visSpec, additionalInfo), renderType: 'tree' domId: 'Accessible-Vis' }) }) ...
